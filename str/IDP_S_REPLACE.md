# 替换字符串的一部分

## 语法

```
s.replace(ss [from] [nc])
```

## 参数

- **s** - str 变量。
- **ss** - 替换字符串。可以是字符串或数值表达式。
- **from** - 替换位置，**s** 中的基于 0 的字符索引。默认值：0。
- **nc** - 要替换的字符数。默认值：-1（从 **from** 开始的右侧所有字符）。

## 备注

替换 **s** 中的一段字符范围。

## 示例

```cpp
str s = "one two three"
s.replace("four" 4 3)
 // 现在 s 是 "one four three"
```
