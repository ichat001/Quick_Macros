# 插入其他字符串

## 语法

```
s.insert(ss [from] [nc])
```

## 参数

- **s** - str 变量。
- **ss** - 要插入的字符串。可以是字符串或数值。
- **from** - 插入位置，**s** 中的基于 0 的字符索引。默认值：0。
- **nc** - 要复制的字符数。默认值：-1（整个 **ss**）。

## 备注

将字符串 **ss** 插入到 **s** 中。

## 示例

```cpp
str s = "My dog cat"
s.insert("and " 7)
 // 现在 s 是 "My dog and cat"
```
